## Introduction
apero-ios-sdk is an library contains integrate Firebase, Adjust, Facebook ... written in Swift.

## Usage
Only one line code to connect Firebase, Adjust, Facebook, ...
```
AperoApp.configure()
```

- [x] Integrate Adjust
- [x] Integrate Facebook
